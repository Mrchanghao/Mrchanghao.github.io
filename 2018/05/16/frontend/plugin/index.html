soundCloud

사실 가장 끌렸다. 블로그를 만들면서 따로 음악 공간을 만들 정도로 일렉음악에 관심이 많은데 명곡들을 포스팅 하고 싶었다. 특히 우리나라에서는 일렉음악이 그냥 클럽음악으로만 치부되는 게 너무 싫은 내 자신으로서는 특히 더 마음에 드는 기능 이다.

npm install –save hexo-tag-soundcloud

youtube

유툽도 그냥 url 정도로 여겼는데 url에서 뒤쪽 id를 따로 떼줘야 한다. 아이디가 없으면 태그 자체는 불러 올 수 있지만 영상 소스를 불러 오지는 못 한다. url이 아니라 재생 id이다.

정리하며

사실 더 많은 플러그인이 있고 더 많이 설치를 해봤다. 그런데 로컬로 실행만 해봤을 뿐이데 마크다운 문법이 깨지거나 갑자기 다른 플러그인이 실행이 안 되는 경우들도 많아서 난 정보 공유와 내 공부 목적으로 하는 블로그에서는 이 정도 플러그인 정도면 만족한다. 그리고 내부 코드블록의 백그라운드와 color를 바꿨다.
이건 플러그인이 아니라 테마 폴더의 소스를 건드려야 해서 조금 조마조마 했지만 그래도 나름 내 생각 대로 나와서 다행이다. 다음에는 폰트를 변경해봐야 겠다.
