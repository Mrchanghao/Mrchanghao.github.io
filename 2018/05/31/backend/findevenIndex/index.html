find Even Index

오늘의 코드워 문제는 날 계속 괴롭혔다. 풀 것 같은데 계속 풀리지가 않고 어떻게 코드를 작성해서 샘플 테스트에서 통과를 해도 전체 랜덤 테스트에서는 택도 없이 틀려 버리기가 일 수 였다. 결국…. 답을 보고야 말았다… 안 그러면 오늘 계속 밤을 샐 것 같아서…ㅠㅠ

일단 문제는 다음과 같다.
어레이를 인수로 주어지면 그 어레이의 오른쪽 합과 왼쪽 합이 같아지는 어레이의 인덱스를 찾는 것인데 예를 들어
[1, 2, 3, 4, 3, 2, 1]라는 어레이가 있으면 왼쪽 합 6, 오른쪽 합 6이 같아지는 어레이의 인덱스 3을 리턴 값으로 받아야 한다는 문제이다. 

function findEvenIndex(arr)
{
  //Code goes here!
   for(var i = 1; i < arr.length; i++) {
        var left = arr.slice(0, i).reduce((a, b) => {
            return a + b;
        }, 0)
        var right = arr.slice(i + 1, arr.length).reduce((a, b) => {
            return a + b;
        }, 0);
        if(left === right) return i
    }
    return -1;
}

정리

처음에 정답을 보기 전에는 for문을 사용하여 활용할 생각을 하지도 못 했다. 다음 부터는 어레이 관련 문제가 나오면 for문 부터 생각을 해봐야 겠다. 물론 칭찬할 점은 간단한 덧셈에 reduce 함수를 이해해서 활용하기 시작했다는 점이다. 물론 이것도 간단한 덧셈이겠지만… 일단 처음이 중요한 것이니까… 다른 분들의 더 훌륭한 코드를 보기 시작했지만 오늘 문제는 내 스스로 푼 것이 아니라 일단 이 문제에 대해 더 생각해 보고 다시 풀어 봐야 겠다.
