오늘의 코드워

코드워에서 메일로 문제가 와서 심심해서 풀어봤다. 역시 코드워의 인공지능님이 내 수준을 헤아려 주셔서 그런지 어려운 문제가 오지 않아 쉽게 풀 수 있는 문제였다.

어레이의 숫자들 중에 양수들의 합만 구하는 문제였다. 뭐 이거야 if문과 for문만 돌리면 금방이니까 알고리즘 코드는 금방 짤 수 있겠다.

코드워에서 샘플 테스트를 실행 해 보니 금방 통과 완료!!

바로 제출 했다.

천재들의 코드

사실 처음에는 나도 reduce를 사용 해 볼까 했는데 그러려면 사실 시간이 더 오래 걸릴 것 같아서 그냥 for문을 이용했다..ㅋㅋㅋ (핑계가 아니고 지금 나갈 준비 해야 하는데 블로그 쓰고 있는 중….)

오잉? 필터는 생각 못 했다… reduce 함수 블록 안에서만 어케 할 까 고민햇는데… 먼저 filter로 새 어레이 만들고 그 안에서 작업하는게 더 효율적이겠구나…

역시나 천재분들의 코드를 리뷰하면 많은 점을 배울 수 있다.. 그런데 써 먹질 못한다. 아오… .

일단 map, filter, reduce에 익숙해져야 하겠다.

정리

일단 알고리즘 문제들을 푸는 것은 즐겁고 익숙 해지기 시작했는데 시간복잡도나 공간 복잡도에 대한 개념 자체는 블로그 글들을 읽어봐도 영 감이 잡히지 않는다. 결국, 비전공자의 약점이 이런 것인가….
틈틈이 알고리즘 개념에 대한 공부 하면서 자료 수집하고 글로 정리를 해야 되겠다.
