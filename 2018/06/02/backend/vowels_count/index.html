vowels count

오늘의 코드워는 사실 예전에 풀어 봤던 문제였다. 그런데도 불구하고 사실 어떻게 풀지 고민을 오래 해서 조금 가슴이 뜨끔했던 문제 이기도 하다.

정해진 문자열을 인자로 받고 그 문자열에 포함된 영어 모음들의 숫자 갯수를 구하는 문제이다. 이건 사실 ES6 문법을 쓰면 아주 쉽게 구할 수 있다. 물론 정규표현식을 쓰면 한줄로도 가능하다. 하지만 난 아직도 정규 표현식을 어떻게 써야 하는지 가늠이 안 간다. 물론 메서드들은 충분히 이해가 가지만 직접 정규표현식을 작성하는게 너무 어렵다….

일단 반복문을 이용하여 구한 방법이다.

위 방식이 내가 푼 방식이다. 물론 다른 방식들을 천재분들이 많이 제시해 주셨다. 역시 가장 쉬운 방법은 정규표현식이였다 .

난 사실 아래 방식이 더 마음에 들었다.

정리

사실 예전에 풀어 봤던 문제여서 블로그에 정리를 하긴 그랬지만 일단 문제를 푸는데 시간이 너무 오래 걸렸다…
결국 나같은 비전공자에 조금 수학적 머리가 없는 사람에겐 이런 방식이 젤로 좋을 듯하다. 풀고 또 풀고 코드의 작동 방식을 이해해야 지만 완전히 내가 푼 문제가 되는 것 같다.
사실 테스트 코드도 작성해야 하는데 … 내 게으름증이 시작되서 오늘은 문제 푸는 걸로 만족해야 겠다.
